# Course Landing Page

A clean and responsive Course Landing Page website built using HTML and CSS.  
The project includes a modern layout with sections like a hero banner, courses, and a learning features section.



## Live Demo
https://ambigaa.github.io/course-landing-page/



## Features
1. Responsive design  
2. Clean and modern UI  
3. Course cards with hover effects  
4. Hero banner section  
5. What You Learn grid  
6. Simple HTML and CSS code
